Restler

History

Requests you send are automatically saved so that you can come back and review at any time.

Collection

Save and organize your requests in folders and load requests without writing its attributes again.

Postman Support

Import and export Postman collections.

Autocomplete

Restler autocompletes headers with commonly-used values and other fields with available environment variables.

Workspace & Environment Variables

Organize your things in common and switch between various environments to use different values in your requests

Authentication

Authenticate your requests with Basic, Bearer Token, Hawk and Digest Auth.

Insomnia Support

Import and export Insomnia collections.

Cookie

Manage cookies and send them with your request.

Tabs

Tabs allow you to keep multiple requests open at the same time and switch between them without losing data.

WebSocket & SSE

You can send messages to a server and receive event-driven responses without having to poll the server for a reply.
